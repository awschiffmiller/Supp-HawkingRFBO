# Supp-HawkingRFBO
Unrecognized importance of hawking prey capture tactics in a tropical plunge diving seabird. Code and supplemental

Data and code for processing Prey Capture Attempt data and accelerometry data, and making plots.
 
Code:Plots and Results
   - HawkingRFBOsCode.Rmd

Functions:
- AxyCalcs: Function to calculate static and dynamic acceleration
   - Description/function in: axyCalcs.Rd

Data:
- fullPCAdf: Prey capture attempt data collected from video footage and reported in results
   - description in: fullPCAdf.Rd
   - data in: fullPCAdf.Rdata
      
- nest_28_axy: accelerometry data for nest/bird 28 - used for representative examples of accelerometry plots (Figure 1a,b,c) 
   - description in: nest_28_axy.Rd
   - data in: nest_28_axy.Rdata 
      
- Raw BORIS* annotations for all camera tagged birds - Aggregated Observations for each video from each bird (excluding fully obscured videos)
   - BORIS annotations folder
      
- N28_M2: Video footage from bird/nest 28. 30 minutes (2nd 1/2hr segment of video footage)
   - find @: 10.5281/zenodo.19964368

Other:
- Ethogram: Behavioral ethogram used to annotate video footage in BORIS*
   - Hawking RFBO Ethogram.boris


*Friard O, Gamba M. 2016 BORIS: a free, versatile open-source event-logging software for video/audio coding and live observations. Methods in ecology and evolution 7, 11, 1325–1330.(doi:10.1111/2041-210X.12584).
